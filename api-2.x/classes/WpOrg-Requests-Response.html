decode_body()

JSON decode the response body.

public decode_body([bool|null $associative = true ][, int $depth = 512 ], int $options) : array<string|int, mixed>

The method parameters are the same as those for the PHP native json_decode() function.

Parameters
$associative : bool|null = true

Optional. When true, JSON objects will be returned as associative arrays; When false, JSON objects will be returned as objects. When null, JSON objects will be returned as associative arrays or objects depending on whether JSON_OBJECT_AS_ARRAY is set in the flags. Defaults to true (in contrast to the PHP native default of null).

$depth : int = 512

Optional. Maximum nesting depth of the structure being decoded. Defaults to 512.

$options : int

Optional. Bitmask of JSON_BIGINT_AS_STRING, JSON_INVALID_UTF8_IGNORE, JSON_INVALID_UTF8_SUBSTITUTE, JSON_OBJECT_AS_ARRAY, JSON_THROW_ON_ERROR. Defaults to 0 (no options set).
